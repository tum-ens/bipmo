BiPMo – Biogas Plant Model¶

BiPMo (Biogas Plant Model) is a simulation framework for the optimal operation of a biogas plant in a given scenario that works in combination with the simulation environment FLEDGE.

The model is organized and implemented based on the flexible building model CoBMo.

The repository can be found on Github.

Features¶

  • Simulation of biogas production given different parameters
    • Anaerobic digestion based on based on the work in Rieke et al. (2018) and Liebetrau et al. (2015)
  • Optimal plant operation over a given planning horizon based on price timeseries
  • Support of a simple model with constant biogas production and a flexible biogas plant model with flexible feeding regime

Work in progress¶

Please note that the repository is under active development and the interface may change without notice. Create an issue if you have ideas / comments / criticism that may help to make the tool more useful. The repository can be found on Github.
